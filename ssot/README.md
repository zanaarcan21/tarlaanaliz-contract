# SSOT (Single Source of Truth) — Contracts Repo

Bu klasör, **normatif** (canonical) KR registry ve bu repo için **filtered view** dosyalarını içerir.

- `kr_registry.md`: Kanonik KR metinleri (tek kaynak).
- `contracts_ssot.md`: Sadece **contracts** bileşenini ilgilendiren KR’lerin listesi + uygulama notları şablonu.

> Not: Bu repo içinde şemalar/OpenAPI bu SSOT’a göre güncellenir. Çelişki halinde SSOT kazanır.
